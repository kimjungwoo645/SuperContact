# SuperContact 설명서

SuperContact는 연락처 입니다.

##기능
####1.연락처를 추가 할 수 있습니다.
-상대방의 이름,전화번호,이메일,사진등 을 등록 할 수 있습니다.
####2.연락처를 수정 할 수 있습니다.
-상대방의 이름,전화번호,이메일,사진등 을 수정 할 수 있습니다.
####3.연락처를 삭제 할 수 있습니다.
####4.연락처가 이름 순으로 자동정렬됩니다.
